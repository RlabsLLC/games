STEP 0: RETRIEVE FILES

1. Click the [ GITHUB ] link at the top of this page.

2. Browse the repository list to find a game file (e.g., game_name.html).

3. Click the file name to open its preview.

4. Locate the "Download" or "Raw" button (top right of file view) to save it.

Windows / Linux / Mac

1. Ensure the .html file is saved to your drive.

2. Right-click the file.

3. Select "Open With" -> Google Chrome, Firefox, or Edge.

* No installation or internet connection required to play.

Chromebook / ChromeOS

1. Save the file to your "Downloads" folder.

2. Open the "Files" app.

3. Double-click the file to launch it directly in the browser.

Mobile Devices

1. Download the file to your device storage.

2. Use a File Manager app to locate the file.

3. Tap to open. Select your preferred browser if prompted.
